# SWJ PM

Static GitHub Pages frontend and Google Apps Script backend source for SWJ PM.

## Files

- `index.html` - frontend shell
- `app.js` - frontend logic and active Apps Script endpoint
- `style.css` - frontend styles
- `Logo.png` - site logo
- `Code.gs` - Google Apps Script backend source

## Not Included

This public deploy package intentionally excludes local/private project files:

- `.clasp.json`
- `.clasprc.json`
- `HANDOFF.md`
- `MASTER_PROMPT.md`
- AI agent instruction files
- local spreadsheets, PDFs, signature images, and setup notes
